  • About

    SkyNet is a project to visualize aviation information in Japan. Based on the flight schedule obtained in real time, we simulate the current Japanese sky situation.

    SkyNetは日本における航空情報を視覚化するプロジェクトです。リアルタイムに取得したフライトスケジュールをもとに、アクセス時点での日本上空をシミュレートしています。
